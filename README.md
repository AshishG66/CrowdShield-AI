# CrowdShield-AI

CrowdShield-AI is an intelligent platform designed for crowd management, safety, and security using AI/ML services.

## Project Structure

- `frontend/` - User interface for monitoring and control.
- `backend/` - Main backend API services.
- `ai-service/` - Artificial Intelligence and Machine Learning models and inference engines.
- `datasets/` - Training data, test data, and sample inputs.
- `docs/` - Project documentation, API specs, and research papers.
- `README.md` - Main entry point and project description.
